# safari cookie 设置有效期最大为7天

  1. Apple在2019年初的时候发布了ITP 2.1版本。在这个版本中，除了第三方cookie会被删以外，第一方cookie也会在7天之内被删除。

  2. 2019年4月，ITP 2.2版本随Safari 12.2发布。这个版本进一步缩短了第一方cookies的寿命——从7天变成了1天。

注意：ITP 2.2使用了某种智能算法，并不会是所有的第一方cookies都会被在24小时内删掉；ITP只会删掉它认为有问题的、很可能是第三方伪装成第一方的跟踪cookies。

# Chrome SameSite设置

  • Chrome 80 默认将没有设置SameSite设置为SameSite=Lax

    解决方法

    1. 谷歌浏览器地址栏输入：chrome://flags/
    2. 找到：SameSite by default cookies、Cookies without SameSite must be secure
    3. 设置上面这两项设置成 Disable

最近更新：Chrome 91 在 flags 设置已经移除了上述 SameSite 相关设置

解决方法

  1. 浏览器快捷方式修改启动参数 (这种方法在 Chrome 94 之后也会被移除)

Windows

  --disable-features=SameSiteByDefaultCookies,CookiesWithoutSameSiteMustBeSecure
1

Mac os

  open -a "Google Chrome" --args --disable-features=SameSiteByDefaultCookies,CookiesWithoutSameSiteMustBeSecure
1

  2. 避免跨域
