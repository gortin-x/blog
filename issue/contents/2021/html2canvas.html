# html2canvas 截图模糊/canvas 绘制模糊

解决办法：将canvas绘制区域根据设备dpr扩大，再利用css样式将尺寸保持为目标尺寸

html2canvas 解决示例：

// 获取想要转换的 DOM 节点
const dom = document.querySelector('.targetDom');
const domStyles = window.getComputedStyle(dom);

// 获取像素比
const dpr = window.devicePixelRatio;

// 创建自定义 canvas 元素
const canvas = document.createElement('canvas');

// 设定 canvas 元素属性宽高为 DOM 节点宽高 * 像素比
canvas.width = width * dpr;
canvas.height = height * dpr;

// 设定 canvas css宽高为 DOM 节点宽高
canvas.style.width = `${domStyles.width}px`;
canvas.style.height = `${domStyles.height}px`;

// 获取画笔
const context = canvas.getContext('2d');

// 将所有绘制内容放大像素比倍
context.scale(dpr, dpr);

// 将自定义 canvas 作为配置项传入，开始绘制
html2canvas(dom, {
    canvas, 
    background: '#ffffff'
}).then((img) => {
    // do something
})

# html2canvas 截图出现空白不完整

原因：出现这种问题是由于滚动造成的，html2canvas是根据屏幕截图进行截图，如果内容高度高于屏幕时，就会出现这样的问题。

解决方法：

  1. 利用fixed布局去除滚动;
  2. 参考https://www.cxyzjd.com/article/chriscencen/109044928 (opens new window)
